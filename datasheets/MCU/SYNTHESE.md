# Synthese rapide - MCU

PDF archive: stm32f427ag.pdf
Reference imprimee: ST STM32F429NIH6 [confirme]
Source officielle attendue: STMicroelectronics (famille STM32F42x/43x)
Role: MCU principal (controle camera, laser, bluetooth, logique metier).

Fonctionnalites exploitables:
- DCMI pour interface camera parallele.
- FMC pour SDRAM externe et gros tampons.
- USB OTG, UART, SPI, I2C pour I/O et connectivite.
