# Synthese rapide - Module Bluetooth

PDF archive: WT12 Data Sheet - WT12-DataSheet.pdf
Reference imprimee: Bluegiga / Silicon Labs WT12-A [confirme]
Source officielle: Silicon Labs / Bluegiga WT12 datasheet
Role: Liaison Bluetooth Classic serie entre STM32 et PC.

Fonctionnalites exploitables:
- Pont UART <-> Bluetooth SPP pour telemetrie/commande.
- Portee et debit adaptes a un outil atelier.
- Integration simple avec pile serie existante cote MCU.
